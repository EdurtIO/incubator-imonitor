#### 二次开发

---

- 安装外部依赖包

```bash
cnpm install -g less
```
或者
```bash
npm install -g less
```

- 克隆源码

```bash
git clone https://github.com/EdurtIO/incubator-imonitor.git
```

- 安装服务依赖

```bash
cd incubator-imonitor
pip install -r agent/src/main/python/requirements.txt
```

- 启动服务

```bash
python agent/src/main/python/Application.py
```

- Web UI

```bash
http://localhost:5000
```
